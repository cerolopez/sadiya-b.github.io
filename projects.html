01 ERROR CHECKER
Java, Spring boot, thymeleaf, MySQL

A website to help the project managers in finding errors in client-given files. This application automates the process of finding errors and points out the cell in the excel workbook that has incorrect data. First, the formatting errors such as char limit, text format example, date, email, etc. are checked, and secondly, data of the required fields are looked up in the database.

Python, RASA, Twilio, SQLite

This project was created as a part of natural language processing coursework.
